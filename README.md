# Splitly

---

## Features

- Create and manage multiple expense groups
- Track participants and group expenses
- Real-time summary of how much each person owes
- Custom currency support (USD, EUR, INR, etc.)
- Share results via link or QR code
- Expense history and settlement checklist

---

## How It Works

1. **Create Group**  
   Enter a name and create an expense group.

2. **Add Participants**  
   List all people participating in the group.

3. **Add Expenses**  
   Add expense records, who paid, and amounts.

4. **Split and Share**  
   See per-person totals and share via link or QR.
